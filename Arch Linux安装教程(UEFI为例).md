# Arch Linux安装教程(UEFI为例)
UEFI与GPT分区表方案：
                    
|   挂载点   |       分区       |        分区类型        |                 建议大小                  |
| :-------: | :-------------: | :------------------: | :--------------------------------------: |
| /mnt/boot | /dev/efi_system |      EFI系统分区       | 至少300MB,如果需要安装多个内核，那就至少1GB |
|  [SWAP]   |      /swap      | Linux swap(交换分区)  |   大于512MB，或者根据计算机的内存大小决定   |
|   /mnt    |    /dev/root    | Linux x86-64根目录(/) |                剩余的空间                 |
